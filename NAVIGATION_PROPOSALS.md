# ナビゲーション改善案

## 📋 現状の課題

- 13個のタブが横スクロールで並んでいる
- スクロールバーが非表示なので、右に機能があることに気づきにくい
- モバイルで操作しづらい
- どこに何があるか探しにくい

---

## 🎯 改善案

### 案1: カテゴリー別2段タブ

**概要:**
機能をカテゴリーで分類し、1段目でカテゴリー選択、2段目で具体的な機能を選択

**カテゴリー分類例:**
```
📊 買い物 (4機能)
  - 単価比較
  - 割引計算
  - 税込/税抜
  - ポイント還元

💰 貯蓄・投資 (3機能)
  - 投資シミュレーター
  - 貯金目標
  - ふるさと納税

💼 給与・収入 (2機能)
  - 時給換算
  - 手取り給与

🏠 ローン・サブスク (3機能)
  - 住宅ローン
  - サブスク管理
  - 移動コスト

🍽️ 生活 (1機能)
  - 割り勘
```

**メリット:**
- 機能が整理されて見つけやすい
- スクロール不要
- カテゴリーが明確で直感的

**デメリット:**
- 2クリック必要
- 画面の縦スペースを多く使う

---

### 案2: ドロップダウンメニュー

**概要:**
「計算ツールを選択」というドロップダウンメニューで全機能を表示

**UIイメージ:**
```
┌─────────────────────────────────┐
│ 💼 単価比較 ▼                   │
├─────────────────────────────────┤
│ 🏷️ 単価比較                     │
│ 💰 割引計算                     │
│ 🧾 税込/税抜                     │
│ 🍽️ 割り勘                       │
│ 💳 ポイント還元                 │
│ 📈 投資シミュレーター           │
│ ... (続く)                      │
└─────────────────────────────────┘
```

**メリット:**
- 省スペース
- 全機能が1箇所で見える
- 検索機能を追加しやすい

**デメリット:**
- 開かないと他の機能が見えない
- タブより操作性が劣る
- モバイルでドロップダウンは使いにくい

---

### 案3: ハンバーガーメニュー + サイドバー

**概要:**
左上のハンバーガーアイコンをクリックすると、サイドバーが開いて全機能を表示

**UIイメージ:**
```
┌──────────────────────────┐
│ ☰  Money Toolbox         │
└──────────────────────────┘
┌─────────┬────────────────┐
│ 機能一覧 │ [コンテンツ]  │
│         │                │
│ 買い物   │                │
│ ├単価比較│                │
│ ├割引計算│                │
│ ...     │                │
│         │                │
│ 貯蓄・投資│               │
│ ├投資   │                │
│ ...     │                │
└─────────┴────────────────┘
```

**メリット:**
- 階層構造で整理しやすい
- モバイルアプリの標準パターン
- スペース効率が良い

**デメリット:**
- メニューを開く操作が必要
- PWAらしさが薄れる
- 実装が複雑

---

### 案4: グリッド表示（ホーム画面）

**概要:**
最初にカード型のグリッドで全機能を表示し、選択すると計算画面へ遷移

**UIイメージ:**
```
┌─────────┬─────────┬─────────┐
│ 🏷️      │ 💰      │ 🧾      │
│ 単価比較 │ 割引計算│税込/税抜│
└─────────┴─────────┴─────────┘
┌─────────┬─────────┬─────────┐
│ 🍽️      │ 💳      │ 📈      │
│ 割り勘  │ポイント │ 投資    │
└─────────┴─────────┴─────────┘
...
```

**メリット:**
- 全機能が一目で見える
- タップしやすい
- 視覚的に美しい

**デメリット:**
- ホーム画面とツール画面の2画面構成になる
- 戻る操作が必要
- スクロールが長くなる

---

### 案5: タブ + 「もっと見る」ボタン

**概要:**
人気の6-8機能はタブで表示、残りは「...」ボタンでモーダル表示

**UIイメージ:**
```
┌────┬────┬────┬────┬────┬────┬───┐
│単価│割引│税込│割勘│ポイ│投資│...│
└────┴────┴────┴────┴────┴────┴───┘
                           ↓クリック
┌─────────────────────────────────┐
│ その他の機能                     │
├─────────────────────────────────┤
│ ⏱️ 時給換算                     │
│ 💰 貯金目標                     │
│ 📱 サブスク管理                 │
│ 🚗 移動コスト                   │
│ 🏠 住宅ローン                   │
│ 💴 手取り給与                   │
│ 🎁 ふるさと納税                 │
└─────────────────────────────────┘
```

**メリット:**
- よく使う機能にすぐアクセス
- スクロール不要
- 既存のタブUIを活かせる

**デメリット:**
- 残りの機能へのアクセスが1クリック増える
- どの機能を優先するか判断が必要

---

### 案6: 検索バー付きタブ

**概要:**
タブの上部に検索バーを配置し、機能名で絞り込める

**UIイメージ:**
```
┌─────────────────────────────────┐
│ 🔍 計算ツールを検索...          │
└─────────────────────────────────┘
┌────┬────┬────┬────┬────┬────┐
│単価│割引│税込│割勘│ポイ│投資│→
└────┴────┴────┴────┴────┴────┘
```

**メリット:**
- 機能が増えても対応可能
- キーワードで素早く見つけられる
- 既存UIを維持できる

**デメリット:**
- 検索する手間がかかる
- 機能を知らないと検索できない
- モバイルでキーボード入力は面倒

## 🎯 推奨案

### 第1推奨: **案1 - カテゴリー別2段タブ**

**理由:**
- 機能が整理されて見つけやすい
- モバイルでもタップしやすい
- カテゴリーは4つなので1段目は余裕で収まる
- PWAらしいシンプルなUI

**実装イメージ:**
1. 1段目: カテゴリータブ（4つ）
   - 📊 買い物・節約
   - 💰 貯蓄・投資
   - 💼 給与・ローン
   - 🍽️ その他

2. 2段目: 各カテゴリー内の機能タブ
   - 選択されたカテゴリーの機能のみ表示
   - 4個以下なので横スクロール不要

---

### 第2推奨: **案3 - ハンバーガーメニュー + サイドバー**

**理由:**
- モバイルアプリの標準的なパターン
- カテゴリー階層で整理しやすい
- 将来的に機能が増えても対応可能
- 最近使った機能の履歴表示も可能

**実装イメージ:**
- 左上にハンバーガーメニュー
- サイドバーに機能を階層表示
- LocalStorageで最近使った機能を記録し、上部に表示

---

### 第3推奨: **案5 - タブ + 「もっと見る」**

**理由:**
- 既存のタブUIを活かせる
- 実装が比較的簡単
- よく使う機能（単価比較、割引計算、税込/税抜など）にはすぐアクセス

**実装イメージ:**
- 最初の6つのタブを表示
- 最後に「...」ボタン
- クリックでモーダルに残りの7機能を表示

---

## 💡 ハイブリッド案（最推奨）

**案1と案5を組み合わせ:**
1. カテゴリー別2段タブを基本とする
2. さらに「最近使った機能」を1段目の左端に追加
3. LocalStorageで履歴を保存

**メリット:**
- リピーターは最近使った機能から1タップでアクセス
- 新規ユーザーはカテゴリーから探せる
- 最大限使いやすい

```
┌────────┬────────┬────────┬────────┐
│⏱最近   │📊買い物│💰投資  │💼給与  │
└────────┴────────┴────────┴────────┘
┌────┬────┬────┬────┐  ← 選択されたカテゴリーの機能
│単価│割引│税込│ポイ│
└────┴────┴────┴────┘
```
